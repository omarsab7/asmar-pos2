# ☕ تشغيل نظام Asmar POS — أبسط طريقة (بدون أي أوامر)

النظام تطبيق حقيقي بقاعدة بيانات، فما بينرفع بالسحب متل الموقع. بس صار يجهّز حالو لحالو — كل يلي عليك نقرات لمرة وحدة.

## بتحتاج 3 حسابات مجانية (10 دقايق)
1. **GitHub** — github.com
2. **Neon** (قاعدة البيانات) — neon.tech
3. **Vercel** أو **Netlify** (الاستضافة)

> أنا رح اعطيك **Vercel** لأنّو الأسهل (بنقرات أقل). إذا مصمّم ع Netlify، نفس الفكرة وملف `DEPLOY_NETLIFY.md` فيه التفاصيل.

---

## الخطوة 1 — قاعدة البيانات (Neon)
1. سجّل على **neon.tech** ← New Project.
2. بيطلعلك سطر طويل اسمو **Connection string** (بيبدأ بـ`postgresql://`). انسخو واحفظو بمكان.

## الخطوة 2 — حطّ الكود ع GitHub
1. سجّل على **github.com** ← اضغط **New repository** ← سمّيه `asmar-pos` ← Create.
2. بصفحة الريبو الفاضية اضغط **uploading an existing file**.
3. فكّ ضغط ملف `asmar-system.zip` على جهازك، وافتح فولدر `asmar-system`، واسحب **كل اللي جوّاتو** ع صفحة GitHub.
4. اضغط **Commit changes**.

## الخطوة 3 — انشر على Vercel
1. سجّل على **vercel.com** بحساب GitHub.
2. **Add New → Project** ← اختار ريبو `asmar-pos` ← **Import**.
3. قبل **Deploy**، افتح **Environment Variables** وحط 3 سطور:
   - `DATABASE_URL` = سطر Neon يلي نسخته
   - `NEXTAUTH_SECRET` = أي كلمة طويلة عشوائية (مثلاً: `asmar_secret_8f3k2j9x7q1`)
   - `NEXTAUTH_URL` = خلّيها فاضية هلّق، بترجعلها بعد ما ياخد دومين (أو حط `https://asmar-pos.vercel.app`)
4. اضغط **Deploy**. بياخد دقيقتين-تلاتة ويجهّز كل شي لحالو (الجداول + البيانات).

## مميزات أضيفت (الزباين والتقارير)
- **الزباين** 👥: ملف لكل زبون (اسم/موبايل/ملاحظة) + سجلّ كل طلباتو وكم صرف. وبتقدر تربط الزبون بالبيع من شاشة الكاشير.
- **التقارير** 📈: اختار المدة (اليوم/أسبوع/شهر/**سنة**) وبتشوف الإيراد والربح وعدد الفواتير + المبيعات يوم بيوم + الأكثر مبيعاً. كل بيع بينحفظ للأبد، فبتقدر تطلّع ربح أي يوم أو السنة كاملة.

## الخطوة 4 — جرّب 🎉
- بيعطيك رابط متل `https://asmar-pos.vercel.app`.
- سجّل دخول: **admin@asmar.coffee** / **admin123**.
- على الهاتف: بيطلع زر **تثبيت** (أو من Safari «أضِف للشاشة الرئيسية») ← بيشتغل متل تطبيق.

---

## إذا بدك Netlify بالضبط
نفس الخطوة 1 و2. بعدها:
1. **app.netlify.com → Add new site → Import an existing project → GitHub → asmar-pos**.
2. **Environment variables**: نفس الـ3 سطور فوق.
3. **Deploy**. (أمر البناء جاهز بملف `netlify.toml`.)

## وقعت بمشكلة؟
- خطأ بالبناء غالباً = `DATABASE_URL` غلط أو ناقص → صحّحو ← Redeploy (بـVercel: Deployments ← Redeploy / بـNetlify: Clear cache and deploy).
- بدّك تستعمل رابط Neon **pooled** (يلي فيه `-pooler`) لـ`DATABASE_URL` حتى ما يعلّق.

— محتاج مساعدة بأي خطوة؟ خبّرني وين وقفت وأنا بمشّيك.
